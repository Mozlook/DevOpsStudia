Zarządzaj produktami i sprawdzaj statystyki backendu

Prosty dashboard oparty wyłącznie na statycznych plikach HTML, CSS i czystym JavaScript. Dane pobierane są z API wystawionego za Nginx pod ścieżką /api/.

Zakres demo

  • Statyczny frontend bez frameworków
  • Lista produktów pobierana przez fetch
  • Formularz dodawania nowych produktów
  • Widok statystyk backendu
  • Integracja przez Nginx z proxy pod /api/

Endpointy używane przez frontend

API

  • GET /api/items — pobranie listy produktów
  • POST /api/items — dodanie nowego produktu
  • GET /api/stats — pobranie statystyk aplikacji
